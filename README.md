# Github Search App

#### This project is deployed to [Netlify](https://resplendent-churros-75d1c6.netlify.app/).

To run locally:
```bash
  yarn install
  yarn start
```
To Build locally:
```bash
  yarn build
```

To Deploy 
```bash
  yarn deploy
```

To run tests:
```bash
  yarn test
```
